# css-to-stylus
Atom package which convert your CSS to Stylus.

## Install

`cmd+,` > Install > 'css-to-stylus'

## Use

Select the text
* right click > 'Convert CSS to Stylus'
* Packages > 'CSS to Stylus' > 'Convert CSS to Stylus'
* ctrl+alt+o

> files which you changing must be `.css` or `.styl`
